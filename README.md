# medical-diagnostics-website
Дипломный проект


## Исползуемые технологии
  * python 3.12
  * postgres

## Инструкция по развертыванию:
  
- В терминале ввести команду для клонирования репозитория
  ```
  git clone https://github.com/
  ```
- Создать виратульное окружение
```
  python -m venv venv
  ```
